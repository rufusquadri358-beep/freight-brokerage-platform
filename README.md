# Freight Brokerage Platform

A decentralized freight brokerage smart contract built on Stacks blockchain that coordinates shipping operations between shippers and carriers.

## Key Features

* **Load Management** - Shippers can post freight loads with detailed specifications
* **Carrier Registration** - Transport companies register with credentials and insurance info
* **Rate Negotiation** - Carriers submit bids and shippers accept competitive rates
* **Delivery Tracking** - Real-time location updates with optional proof-of-delivery hashing
* **Status Management** - Complete workflow from load posting to delivery confirmation
